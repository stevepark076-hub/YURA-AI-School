## ⚠️ Task 2 — Supabase 수동 작업 (완료 전까지 Task 3 진행 불가)

### 1. Supabase 프로젝트 생성
- https://supabase.com → New Project
- 프로젝트명: `yura-ai-school`

### 2. SQL Editor에서 스키마 실행
`docs/plans/2026-03-16-yura-ai-school-enrollment.md` → Task 2 섹션의 SQL 전체 복사 → 실행

생성되는 테이블:
- `profiles` — 사용자 정보 (사번, 이름, 부서, 역할)
- `courses` — 강좌
- `course_sessions` — 회차 (날짜, 장소, 정원 등)
- `enrollments` — 수강신청

### 3. 관리자 계정 생성
- Supabase 대시보드 → Authentication → Users → Add user
  - Email: `admin@yura.internal`
  - Password: 원하는 비밀번호

### 4. profiles 테이블에 관리자 등록
```sql

  '위에서_생성된_USER_UUID',

  '인사팀',

```

### 5. .env.local 환경변수 입력
Project Settings → API에서 복사:
```

SUPABASE_SERVICE_ROLE_KEY=eyJ...
```
